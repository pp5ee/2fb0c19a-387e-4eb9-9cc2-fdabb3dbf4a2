# 贪吃蛇游戏 (Snake Game)

一个使用 HTML、CSS 和 JavaScript 实现的经典贪吃蛇网页游戏。

## 功能特点

- 🎮 经典贪吃蛇玩法
- ⌨️ 方向键控制蛇的移动
- ⏸️ 空格键暂停/继续游戏
- 🏆 最高分记录（自动保存到本地）
- 📊 实时分数显示
- 🎨 现代界面设计

## 如何运行

直接在浏览器中打开 `snake.html` 文件即可开始游戏。

```bash
# 使用浏览器打开
open snake.html
# 或者使用任何本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000/snake.html
```

## 游戏操作

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 控制蛇的移动方向 |
| 空格键 | 暂停/继续游戏 |
| 重新开始按钮 | 重新开始游戏 |

## 游戏规则

- 蛇吃到红色食物后长度增加，分数+10
- 撞到墙壁或自身，游戏结束
- 最高分自动保存，刷新页面后仍可查看

## 技术实现

- **渲染**: HTML5 Canvas API
- **状态存储**: localStorage API
- **样式**: 纯 CSS，实现现代暗色主题
- **无外部依赖**: 纯原生实现，无需任何框架
